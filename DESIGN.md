# KAIROS 行情筛选器 - 设计思考

## 产品定位

KAIROS 是一个加密货币行情筛选器，帮助用户在海量代币中快速发现"概率上最有潜力"的标的。产品气质：**赛博朋克操盘台**。

## 视觉策略

### 核心意象
> "深夜的交易终端，屏幕的冷光映照着操盘手的脸。数据如瀑布般流动，每一个数字都是决策的锚点。"

### 配色方案
| 角色 | 色值 | 用途 |
|------|------|------|
| **主背景** | `#0A0A0F` | 纯黑，沉浸式数据舞台 |
| **次级背景** | `#12121A` | 卡片、列表项背景 |
| **霓虹青** | `#00F0FF` | 上涨信号、买入信号 |
| **霓虹紫** | `#BF00FF` | 下跌信号、卖出信号 |
| **霓虹金** | `#FFD700` | 强调、热门标记 |
| **文字主** | `#FFFFFF` | 主要数据展示 |
| **文字次** | `#6B7280` | 辅助说明文字 |
| **边框线** | `#1F1F2E` | 分隔线、低调装饰 |

### 字体与排版
- **数据字体**：等宽字体 (JetBrains Mono) - 数字更清晰
- **标题字体**：粗体无衬线 (Inter Bold)
- **字号策略**：大数字 + 小说明，极致对比

### 动效设计
- **数字跳动**：关键数据变化时使用轻微缩放动画
- **渐变流动**：背景使用缓慢的渐变动画营造"数据流"感
- **卡片入场**：轻微的上浮 + 淡入，按序延迟

## 组件设计

### 场景卡片
- 大圆角 (24px)
- 霓虹色渐变边框
- 内部微发光效果
- 箭头图标 + 文字组合

### 代币列表项
- 卡片式布局
- 左侧：币种图标 + 名称 + 符号
- 右侧：价格 + 涨跌幅
- 下方：关键指标网格展示
- 上涨用霓虹青，下跌用霓虹紫

- VIP 专属功能入口
- 会员标识（金色皇冠图标）
- 2x2 网格布局展示快捷功能
- 每项包含：图标、标题、副标题描述
- 整体用霓虹金 (#FFD700) 作为强调色

### 指标徽章
- 小圆角胶囊形状
- 半透明背景 + 边框
- 数值用等宽字体
- 颜色与指标状态关联

## 技术指标展示规范

| 指标 | 显示格式 | 颜色逻辑 |
|------|----------|----------|
| RSI | `RSI: 65` | 50-70 正常白，<40 紫(弱)，>80 青(强) |
| ADX | `ADX: 32` | >25 正常白，>40 青(强趋势) |
| 成交量比 | `量: 1.8x` | >1.5 青(放量)，<0.8 紫(缩量) |
| MFI | `MFI: 72` | >60 青(资金流入)，<40 紫(流出) |
| EMA | `EMA50↑` | 价格站上EMA显示青色 |
| MACD | `MACD: +` | 零轴上方青色，下方紫色 |

## 设计禁忌
1. **禁止**使用纯白/浅色主题 - 违背产品的专业金融感
2. **禁止**使用彩色渐变背景 - 数据需要高对比度可读性
3. **禁止**使用圆润可爱的图标 - 保持科技硬核感
4. **禁止**过度动画 - 数据展示需要稳定，不干扰阅读
